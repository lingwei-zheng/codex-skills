# Evidence Triangulation

Use this before making novelty, literature-gap, field-trend, or journal-fit claims.

## Evidence Order

1. User-provided proposal, manuscript, journal list, figures, tables, and notes.
2. Zotero local library: known papers, saved PDFs, tags, collections, citation metadata. Follow `zotero-evidence-workflow.md` for the execution protocol.
3. Web literature search: recent papers, review articles, methods papers, journal scopes, author guidelines, special issues. Use this when Zotero is unavailable, sparse, outdated, or insufficient for the decision.
4. Inference from evidence: allowed only when labeled.
5. Unknown: leave unresolved when evidence is missing.

## Zotero Search Pattern

Start with the Zotero workflow file, then search with:

- Core concepts and synonyms from the research question.
- Method terms and data source names.
- Target journal names if a journal list is supplied.
- Key constructs plus geography, GIScience, health geography, mobility, exposure, environment, or spatial analysis terms when relevant.

Use Zotero full text only when metadata is insufficient and the claim matters. If Zotero cannot be used, record the blocker in the ledger before relying on web search.

## Web Search Pattern

Browse when:

- The user asks for latest/current literature.
- Journal guidelines, scope, or recent article shape matters.
- Zotero has sparse evidence.
- A novelty or gap claim would decide the recommendation.

Prefer primary sources: journal pages, publisher author guidelines, paper pages, DOIs, databases, and official issue pages.

## Claim Discipline

- Do not turn "I found little evidence" into "no literature exists."
- Do not use citations as decoration; each cited source must support the claim it is attached to.
- If evidence conflicts, report the conflict and say how it affects the recommendation.
- If a decisive claim is not verified, keep the journal recommendation provisional.
