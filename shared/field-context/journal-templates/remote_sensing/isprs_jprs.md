# ISPRS Journal of Photogrammetry and Remote Sensing
## Paper Template & Formatting Guide

**Publisher**: Elsevier / ISPRS
**Impact Factor**: ~12
**Scope**: Photogrammetry, remote sensing, spatial information science, computer vision for geospatial

---

## Submission Requirements

| Requirement | Specification |
|-------------|--------------|
| Word limit | 12,000 words |
| Abstract | 250 words maximum |
| Keywords | 5–8 |
| Highlights | 3–5 bullets (≤85 chars) |
| References | Numbered Vancouver style |
| Figures | 300 DPI, TIFF/PDF |
| Graphical abstract | Recommended |

- Sensor specifications (focal length, pixel size, flight height for photogrammetry; orbit, band specs for satellite)

### 4. Methodology
- Mathematical formulation
- Algorithm description with pseudocode
- Implementation details

### 5. Experiments and Results
- Quantitative accuracy assessment
- Visual/qualitative assessment

### 6. Discussion
- Error analysis and sources
- Comparison with published benchmarks
- Limitations and future work

### 7. Conclusion

---

## ISPRS-Specific Notes

- **Photogrammetry papers**: include point cloud density, reconstruction accuracy
- **Classification papers**: per-class F1, OA, Kappa, confusion matrix
- **Change detection**: include temporal baseline, change commission/omission errors
- Reference **ISPRS benchmark datasets** where applicable

## Checklist

- [ ] Sensor/acquisition parameters complete
- [ ] Independent accuracy assessment
- [ ] Confusion matrix or equivalent
