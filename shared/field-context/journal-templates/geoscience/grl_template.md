# Geophysical Research Letters (GRL)
## Paper Template & Formatting Guide

**Publisher**: AGU / Wiley
**Impact Factor**: ~5
**Scope**: High-impact, concise letters on geophysics, Earth science, space science

---

## Submission Requirements

| Requirement | Specification |
|-------------|--------------|
| Word limit | 4,000 words (text only, excl. references) |
| Abstract | 150 words maximum |
| Figures | ≤5 figures |
| Tables | ≤1 table (additional in supporting info) |
| References | AGU style |
| Supporting info | Encouraged for data, methods details, additional figures |

---

## GRL Letter Structure

### Plain Language Summary (≤150 words)
**Required.** Non-technical summary for general public. No jargon. Explain "what, why, and so what".

### Abstract (≤150 words)
No citations. Present the single most important finding immediately. GRL is a letters journal — lead with the result.

### Key Points (exactly 3)
```
• [Finding 1 — quantitative if possible] (≤140 chars)
• [Finding 2 — mechanism or process] (≤140 chars)
• [Finding 3 — implication or significance] (≤140 chars)
```

### 1. Introduction
**2 paragraphs maximum.**
- Para 1: Why this is important and timely (cite recent papers)
- Para 2: What this study does and its advance

### 2. Data and Methods
Concise. Details go in Supporting Information.
- Data sources with references/DOIs
- Key method parameters only
- Direct readers to Supporting Information for full derivations

### 3. Results
Lead with the main finding. Use quantitative language.
- Each figure should tell a self-contained story
- Maximum 5 figures total in main text

### 4. Discussion and Conclusions
**1–2 paragraphs.**
- Physical interpretation
- Implications for the field
- Outstanding questions

### Open Research
Data availability, code availability, supporting information.

---

## GRL-Specific Rules

- **Lead with the result**: GRL readers want to know the finding immediately
- **5-figure maximum**: Be ruthlessly selective; move details to Supporting Info
- **Supporting Information**: Use it aggressively — full methods, extra figures, raw data
- **Plain Language Summary**: Non-negotiable; often used for press releases
- **Fast publication**: GRL is known for speed; prepare supplementary materials carefully

## Checklist

- [ ] Abstract ≤150 words, leads with main finding
- [ ] Exactly 3 Key Points, each ≤140 characters
- [ ] Plain Language Summary ≤150 words
- [ ] ≤5 figures in main text
- [ ] ≤1 table (rest in Supporting Info)
- [ ] Total text ≤4,000 words
- [ ] Supporting Information prepared
- [ ] Data DOI provided in Open Research section
