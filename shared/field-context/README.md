# Field Context Reference Pack

This directory is a non-triggering reference pack derived from NORA materials.
It is not an installed NORA skill.

Use these files from ARS, peer-review, nature-figure, or conceptual-figure-workflow
when a task needs geography, human geography, GIScience, GeoAI, spatial analysis,
environmental exposure, or health geography grounding.

Default language for Chinese notes and reports is Simplified Chinese unless the
user explicitly requests Traditional Chinese.
