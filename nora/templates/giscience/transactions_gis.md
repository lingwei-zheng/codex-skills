# Transactions in GIS
## Paper Template & Formatting Guide

**Publisher**: Wiley
**Impact Factor**: ~3
**Scope**: Methods, applications, and theory in GIS; human-computer interaction in GIS; spatial data infrastructures

---

## Submission Requirements

| Requirement | Specification |
|-------------|--------------|
| Word limit | 8,000 words |
| Abstract | 150–250 words |
| Keywords | 5–8 |
| References | Wiley APA style |
| Figures | 300 DPI, separate files |
| Review type | Double-blind |

---

## Paper Structure

### Abstract (150–250 words)
Background, purpose, approach, results, conclusions. No citations.

### Keywords (5–8)
At least one geographic/spatial term required.

### 1. Introduction
- GIS application context
- Research gap
- Paper structure paragraph

### 2. Background / Related Work

### 3. Study Area and Data

### 4. Methodology

### 5. Results

### 6. Discussion
- Practical GIS implications
- Generalizability to other areas/scales

### 7. Conclusions

### References
Wiley APA format with DOIs.

---

## TGIS-Specific Notes

- Emphasize GIS implementation details
- Discuss data quality and uncertainty
- Note scalability and computational requirements
- Include workflow diagram if complex

## Checklist

- [ ] Double-blind: remove author info from manuscript file
- [ ] Abstract 150–250 words
- [ ] GIS workflow clearly described
- [ ] Data sources with access information
- [ ] Computational requirements noted
